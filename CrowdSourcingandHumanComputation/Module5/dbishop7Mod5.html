from sklearn.model_selection import train_test_split
from sklearn.linear_model import LogisticRegression
from sklearn.metrics import classification_report
import pandas as pd

# Combine the datasets with a new column 'color' for wine type
red_wine['color'] = 0  # Red wine
white_wine['color'] = 1  # White wine
wines = pd.concat([red_wine, white_wine], ignore_index=True)

# Split the data into features and target variable
X = wines.drop(['color'], axis=1)  # Features
y = wines['color']  # Target variable

#Fill blank variables so training does crash
X = X.fillna(X.median())

# Split the dataset into training and test sets
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.3)

# Task 4: Initialize and train the logistic regression model
model = LogisticRegression(max_iter=1000)
model.fit(X_train, y_train)

# Print the coefficients and intercept
print("Intercept:", model.intercept_)
print("Coefficients:", model.coef_)

# Predict on the test set and print the classification report
predictions = model.predict(X_test)
print(classification_report(y_test, predictions))

from sklearn.ensemble import RandomForestClassifier
from sklearn.model_selection import StratifiedKFold, cross_val_score
from sklearn.metrics import make_scorer, accuracy_score, cohen_kappa_score

# Initialize the Random Forest classifier
rf_classifier = RandomForestClassifier(n_estimators=100)

# Set up Stratified K-Fold cross-validation
skf = StratifiedKFold(n_splits=5, shuffle=True)

accuracy_scorer = make_scorer(accuracy_score)
kappa_scorer = make_scorer(cohen_kappa_score)

# Perform cross-validation for accuracy
accuracy_scores = cross_val_score(rf_classifier, X, y, cv=skf, scoring=accuracy_scorer)
highest_accuracy = max(accuracy_scores)

# Perform cross-validation for kappa
kappa_scores = cross_val_score(rf_classifier, X, y, cv=skf, scoring=kappa_scorer)
average_kappa = kappa_scores.mean()

print(f"Highest Accuracy from cross-validation: {highest_accuracy:.4f}")
print(f"Average Kappa from cross-validation: {average_kappa:.4f}")

Different Models

In this notebook I utilized three different models per instruction, a Logistic Regressor, Random Forest, and a CART model (specifically Decision Tree Classifier). Each model has their pros and cons and can be used for different datasets to varying degrees of effectiveness. So the Logistic Regressor, it was able to perform fairly well at 98% for precision, recall, and f1-scores. This makes sense as Logistic Regressors are made for binary classifiaction, which this is. It seems to have been capable of determining patterns in the data, most likely with the low P score valued items mentioned earlier. Random Forest Classifier was the next model trained on this dataset. It was able to perform fairly well as well getting a highest accuracy score of 99% and a kappa of 98.5%. Again this makes sense as it creates multiple decision trees in which it averages decisions from to help it determine what group something should belong to. Luckily this dataset is fairly small as compared to Logistic Regression it can get a little more computationally expensive (not too bad though in my experience). Finally I trained a Decision Tree Classifier. This is fairly similar to the Random Forest model however it focuses more on making one tree to make decisions instead of averaging many. Much like the Random Forest model it had high scores in everything including accuracy, precision, recall, and f1-score. As this was a fairly simple dataset we didn't need to worry about overfitting too badly and it worked out for this model quite well. Overall, each model was able to perform quite well on this dataset.
